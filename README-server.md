# FTK Shared Server

## تشغيل السيرفر محليًا

```bash
cd doctor/ftk
npm install
npm start
```

افتح المتصفح على:

- `http://localhost:3000`

## الفكرة

- الواجهة `ftkdo.html` ما زالت تعمل كالمعتاد.
- عند وجود السيرفر، البيانات المشتركة تتزامن تلقائيًا بين كل الأجهزة/الحسابات:
  - الحسابات
  - المرضى
  - الشات
  - جدول المواعيد
  - أرشيف المواعيد
- لو السيرفر غير متاح، التطبيق يرجع تلقائيًا للتخزين المحلي بدون توقف.

## ملاحظات النشر

- على GitHub Pages فقط (استضافة ملفات ثابتة) لن يعمل السيرفر، وسيشتغل التخزين المحلي فقط.
- للمزامنة بين كل المستخدمين لازم تنشر `server.js` على منصة تدعم Node.js (مثل Render أو Railway أو VPS).
